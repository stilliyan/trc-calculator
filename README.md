# TRT Calculator

Simple static calculator for converting TRT dose values between mg, mL, and U-100 syringe units.

## Features

- Weekly dose mode
- Single dose mode
- Live mg, mL, and U-100 unit conversion
- Custom injection frequency
- Responsive dark UI
- Bulgarian and English language switch

## Run locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://localhost:4173/
```

## Note

This is a value conversion calculator, not medical advice.
